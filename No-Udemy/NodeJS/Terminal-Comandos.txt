INSTALAR O NPM

(Cria packge.json)
- npm init 
(instala um pacote localmente que só poderá ser usado no projeto especifico em que o comando foi digitado)
- npm install express
Instalar o pacote localmente, logo, poderemos utiliza-lo em todo projeto


Após o "npm-init" é criado um arquivo packge.json, contendo todas as informações

Para executar o node no terminal utilizamos: 
"node "nome do arquivo js"

ls: qual diretorio eu estou
cd: entrar no diretorio

- npm install -g nodemon --save-dev
(instalar globalmente e cria uma página node_modules)
